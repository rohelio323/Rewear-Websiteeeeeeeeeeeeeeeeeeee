# ♻️ ReWear

> **The Living Archive of Fashion.** Preserving the history of craftsmanship through circular fashion and sustainable second-hand marketplaces.

ReWear is a web-based marketplace application built to promote the circular economy. By connecting buyers and sellers of pre-loved clothing, ReWear actively helps reduce fashion waste. A core feature of the platform is tracking the environmental impact of these transactions, displaying the total kg of CO₂ saved by choosing second-hand over fast fashion.

## ✨ Key Features

* **Sustainable Marketplace:** Browse, list, and purchase pre-loved garments.
* **Environmental Impact Tracking:** Real-time calculation and display of CO₂ emissions saved per transaction and across a user's lifetime.
* **User Profiles & Dashboards:** Custom buyer and seller profiles to track orders, listings, and environmental milestones.
* **Admin Dashboard:** Dedicated admin controls for managing the platform, users, and marketplace health.
* **Responsive UI:** A premium, minimalist interface inspired by editorial fashion, fully responsive across all devices.

## 🛠️ Tech Stack

* **Framework:** [Laravel](https://laravel.com/) (PHP)
* **Frontend:** [Tailwind CSS](https://tailwindcss.com/) & Laravel Blade
* **Database:** SQLite (Local Development)
* **Authentication:** Laravel Breeze
